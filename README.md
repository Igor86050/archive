Este projeto foi desenvolvido com o objetivo de praticar análise de dados utilizando Python e bibliotecas voltadas para Data Science.

A base de dados utilizada contém informações de músicas do Spotify, permitindo explorar métricas como:

popularidade,
artistas,
duração das músicas,
rankings,
estatísticas,
visualizações gráficas.

O projeto foi criado como parte do meu aprendizado em:

Python para análise de dados,
Pandas,
Matplotlib,
lógica analítica,
manipulação de datasets.
